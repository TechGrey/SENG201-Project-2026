## Setting up the Game to Play

Setting up the game is incredibly simple. Please follow these steps:

1. Open IntelliJ IDEA.
2. Select "**Clone Repository**".
3. In the **_URL_** field, enter this link: git@eng-git.canterbury.ac.nz:seng201-2026/team-99.git
4. Hit **Clone**.

### Loading the Game
To play the game, use either of these handy-dandy methods:

### From Java File
1. Simply run `java -jar seng201_team99-1.0-SNAPSHOT.jar` to launch the game.

#### From Source code
1. in IntelliJ click the Project menu (the folder in the top left).
2. Navigate to src/main/java/seng201/team99/App.java.
3. Hit the green arrow next to main().
4. Enjoy!

#### From Gradle
1. Open IntelliJ.
2. Open the Terminal pane (bottom left corner, it looks like a rectangle with a right-facing chevron inside it).
3. Run this command: `./gradlew run`
4. The game will open. Enjoy!
